고도 독립형 요소
ElevationElement [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElement]는 단일 지점 또는 다중선의 고도 데이터를 시각적으로 표시하는 HTML 요소입니다. 위도 및 경도 좌표 배열로 path [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElement.path] 속성에 설정된 경로를 따라 해발 높이 그래프를 표시합니다. 선분은 배열의 포인트 간에 보간되므로 소수의 포인트로도 세부적인 고도 그래프를 생성할 수 있습니다.
좌표를 사용하여 경로 고도 설정
다음 예에서는 gmp-elevation 요소를 사용하여 좌표 집합으로 선택한 두 지점 사이의 경로 고도를 렌더링합니다.
요소가 unit-system [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElement.unitSystem] 속성을 사용하여 측정항목 측정에 맞게 구성됩니다.
  <gmp-elevation unit-system="metric"></gmp-elevation>
querySelector는 고도 요소를 선택하고 위도 및 경도 좌표로 path 속성을 설정하는 데 사용됩니다.
/** Maps JS loaded callback */
async function init() {
  // Load the Elevation Element from Maps JS
  const {ElevationElement} = await google.maps.importLibrary('elevation');

  // Specify an elevation path to render
  const elevationElement = document.querySelector('gmp-elevation');
  elevationElement.path = [
    {lat: 37.4523, lng: -122.2645},
    {lat: 37.377, lng: -122.4078},
  ];
}

다중선 객체를 사용하여 경로 고도 설정
다음 예에서는 사용자가 그린 다중선을 기반으로 경로의 고도를 렌더링합니다. polyline.getPath().getArray() [https://developers.google.com/maps/documentation/javascript/reference/polygon?hl=ko#Polyline.getPath]를 사용하여 사용자의 다중선에서 path [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElement.path]를 가져옵니다.
      elevationElem.path = polyline.getPath().getArray();
