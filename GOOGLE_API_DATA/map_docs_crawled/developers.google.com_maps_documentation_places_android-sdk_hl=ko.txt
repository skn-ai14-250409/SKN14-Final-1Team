Android용 Places SDK 상황에 따라 사용자 기기 근처의 지역 비즈니스 및 기타 장소에 응답하는 위치 인식 앱을 빌드합니다.

시작하기

Android용 Places SDK의 일반적인 기능을 사용하여 빌드를 시작합니다.

explore [https://developers.google.com/maps/documentation/places/android-sdk/cloud-setup?hl=ko]

Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/places/android-sdk/cloud-setup?hl=ko]

계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.

settings [https://developers.google.com/maps/documentation/places/android-sdk/choose-sdk?hl=ko]

SDK 버전 선택 [https://developers.google.com/maps/documentation/places/android-sdk/choose-sdk?hl=ko]

Places는 Android용 Places SDK 및 Android용 Places SDK (신규)라는 두 가지 SDK 버전을 지원합니다. 나에게 적합한 버전을 선택하는 방법을 알아보세요.

settings [https://developers.google.com/maps/documentation/places/android-sdk/start?hl=ko]

앱 구성 [https://developers.google.com/maps/documentation/places/android-sdk/start?hl=ko]

Android용 Places SDK를 사용하도록 앱을 구성합니다.

travel_explore [https://developers.google.com/maps/documentation/places/android-sdk/current-place?hl=ko]

현재 장소 찾기 [https://developers.google.com/maps/documentation/places/android-sdk/current-place?hl=ko]

기기의 현재 보고된 위치에서 장소를 탐색합니다.

loupe [https://developers.google.com/maps/documentation/places/android-sdk/place-details?hl=ko#get-place]

ID로 장소 가져오기 [https://developers.google.com/maps/documentation/places/android-sdk/place-details?hl=ko#get-place]

장소 ID를 사용하여 장소에 대한 세부정보를 가져옵니다.

input [https://developers.google.com/maps/documentation/places/android-sdk/place-autocomplete?hl=ko]

장소 예상 검색어 반환 [https://developers.google.com/maps/documentation/places/android-sdk/place-autocomplete?hl=ko]

Place Autocomplete을 사용하여 사용자 검색어에 대한 응답으로 장소 예상 검색어를 반환합니다.

code [https://developers.google.com/maps/documentation/android-sdk/current-place-tutorial?hl=ko]

지도에서 현재 장소를 선택하고 세부정보를 표시합니다. [https://developers.google.com/maps/documentation/android-sdk/current-place-tutorial?hl=ko]

Android 기기의 현재 위치를 찾고 해당 위치에 있는 장소 또는 기타 관심 장소의 세부정보를 표시합니다.

code [https://github.com/googlemaps-samples/android-places-demos]

Places SDK용 데모 앱 [https://github.com/googlemaps-samples/android-places-demos]

GitHub의 Google 샘플 저장소를 방문하여 자바 및 Kotlin용 Android 데모를 설치하고 실행하세요.

code [https://github.com/googlemaps/android-maps-utils/blob/main/demo/src/main/java/com/google/maps/android/utils/demo/HeatmapsPlacesDemoActivity.java]

Places API 검색결과에서 히트맵 생성 [https://github.com/googlemaps/android-maps-utils/blob/main/demo/src/main/java/com/google/maps/android/utils/demo/HeatmapsPlacesDemoActivity.java]

Places API의 검색결과를 사용하여 히트맵을 생성하고 색상 옵션을 확인하며 여러 히트맵을 사용합니다.

기능

Android용 Places SDK의 핵심 기능에 대해 알아보세요.

ID로 장소 가져오기 [https://developers.google.com/maps/documentation/places/android-sdk/place-details?hl=ko#get-place]

장소 ID를 사용하여 장소를 요청합니다.

현재 장소 찾기 [https://developers.google.com/maps/documentation/places/android-sdk/current-place?hl=ko]

기기의 현재 보고된 위치에서 장소를 찾는 방법을 알아봅니다.

Place Autocomplete [https://developers.google.com/maps/documentation/places/android-sdk/place-autocomplete?hl=ko]

애플리케이션에 자동 완성 기능을 추가합니다(장소 이름, 주소 또는 플러스 코드로 검색).

장소 세부정보 [https://developers.google.com/maps/documentation/places/android-sdk/details-place?hl=ko]

특정 시설 또는 관심 장소에 대한 세부정보를 가져옵니다.

텍스트 검색 [https://developers.google.com/maps/documentation/places/android-sdk/text-search?hl=ko]

텍스트 문자열을 기준으로 장소를 검색합니다.

장소 사진 [https://developers.google.com/maps/documentation/places/android-sdk/place-photos?hl=ko]

애플리케이션에 고화질 장소 사진을 추가합니다.

장소 ID [https://developers.google.com/maps/documentation/places/android-sdk/place-id?hl=ko]

특정 장소에 대한 세부정보를 확인하고, 특정 장소의 ID를 찾고, 장소 ID 저장 및 새로고침에 대해 자세히 알아보세요.

장소 유형 [https://developers.google.com/maps/documentation/places/android-sdk/place-types?hl=ko]

장소 유형을 사용하여 장소 검색 및 자동 완성 요청의 결과를 제한합니다.

장소 아이콘 [https://developers.google.com/maps/documentation/places/android-sdk/icons?hl=ko]

다양한 유형의 장소에 대한 아이콘을 요청하고 표시합니다.

유틸리티 및 도우미 라이브러리

Android용 Places SDK의 유틸리티로 앱을 개선합니다.

Kotlin 확장 프로그램을 사용하여 간결하고 자연스러운 Kotlin을 사용 설정합니다.
