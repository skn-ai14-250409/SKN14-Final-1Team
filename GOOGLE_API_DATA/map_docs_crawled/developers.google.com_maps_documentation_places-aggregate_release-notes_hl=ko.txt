This page is updated with each new release of the
Places Aggregate API. The changelog lists releases by date and includes
any new features, bug fixes and significant performance improvements.

May 01, 2025
Announcing the General Availability (GA) release of the Places Aggregate API. See the Release Notes [https://developers.google.com/maps/documentation/places-aggregate/release-notes] for information about this release and for all other releases.

If you are a new user, see Set up your Google Cloud project [https://developers.google.com/maps/documentation/places-aggregate/cloud-setup] to start the installation process.
March 24, 2025
To better align with the product's functionality and value, we changed the product name from Places Insights API to Places Aggregate API.
Note: None of the API methods or objects were renamed so your code should continue to work without change.
February 25, 2025
Places Insights API

Announcing the General Availability (GA) release of the Places Insights API. See the Release Notes [https://developers.google.com/maps/documentation/places-insights/release-notes] for information about this release and for all other releases.

If you are a new user, see Set Up in the Google Cloud Console [https://developers.google.com/maps/documentation/places-insights/cloud-setup] to start the installation process.
For information about how Google calculates pricing and billing for the Places Insights API, see Places Insights API Usage and Billing [https://developers.google.com/maps/documentation/places-insights/usage-and-billing].
September 09, 2024
The Preview release of the Places Insights API is now available. For information about this release, and for all previous releases, see the Release Notes [https://developers.google.com/maps/documentation/places-insights/release-notes] or subscribe to the Release Notes XML feed [https://developers.google.com/static/feeds/places-insights-release-notes.xml].

To start the installation process, see Set Up in the Google Cloud Console [https://developers.google.com/maps/documentation/places-insights/cloud-setup].
