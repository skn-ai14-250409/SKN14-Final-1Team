특성
Directions API의 핵심 기능에 대해 알아보세요.
경로 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko]
한 장소에서 다른 장소로 가는 경로를 요청합니다.
이동수단 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko#TravelModes]
도보, 운전, 대중교통 또는 자전거 길찾기를 하려면 이동 수단을 설정합니다.
경유지 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko#Waypoints]
중간 위치의 정류장을 포함하는 경로를 반환하려면 경유지를 지정하세요.
트래픽 예측 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko#DirectionsAdvanced]
경로의 예상 시간을 반환합니다.
위치 한정자 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko#location-modifiers]
위치 한정자를 지정하여 운전자가 특정 위치에 어떻게 접근해야 하는지를 나타냅니다.
대체 경로 [https://developers.google.com/maps/documentation/directions/get-directions?hl=ko#alternatives]
가능한 경우 여러 경로를 반환하도록 API에 지시합니다.
