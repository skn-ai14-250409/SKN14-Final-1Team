API 선택기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

API 선택도구는 지도 또는 위치 기반 데이터로 수행할 수 있는 가장 일반적인 작업을 나열하고 니즈에 가장 적합한 API를 제안합니다.
각 API에 대한 자세한 내용은 관련 문서 링크를 참조하세요.

| 수행할 작업은 무엇인가? | 유용한 API: |
| --- | --- |
| Android 앱에 지도를 추가합니다. 선택적으로 지도의 3D 카메라를 완전히 제어하여 기울이기, 회전, 확대/축소, 화면 이동을 위한 지도 동작을 지원합니다. | Android용 Maps SDK |
| iOS 앱에 지도를 추가합니다. 선택적으로 지도의 3D 카메라를 완전히 제어하여 기울기, 회전, 확대/축소, 화면 이동을 위한 지도 동작을 지원합니다. | iOS용 Maps SDK |
| 웹페이지에 지도를 추가합니다. 자체 데이터와 이미지를 사용하여 지도를 맞춤설정합니다. 선택적으로 모바일 웹에서 지도 컨트롤과 동작을 지원합니다. | Maps JavaScript API |
| URL과 매개변수로만 구성된 간단한 HTTP 요청을 통해 웹페이지에 지도를 추가합니다. 블로그 및 <script> 태그를 추가할 수 없는 기타 위치에서 유용합니다. | Maps Embed API |
| 웹페이지에 로컬 컨텍스트 지도 추가 로컬 컨텍스트 지도는 지정한 위치 근처의 주요 관심 장소를 사용자에게 보여주는 간단하고 유연한 UI 위젯입니다. 로컬 컨텍스트 라이브러리는 대화형 지도로 구성되며 측면 패널에는 사진 갤러리로 구성된 장소 선택기가 있습니다. 유형을 기준으로 장소 목록을 필터링할 수 있고, 운전 경로를 제공하고, 위젯의 시각적 모양을 맞춤설정할 수 있습니다. | Maps JavaScript API, 로컬 컨텍스트 라이브러리 |
| 지도에 경로를 그립니다. | Android용 Maps SDK의 폴리라인 iOS용 Maps SDK의 폴리라인 Maps JavaScript API의 폴리라인 Maps Static API의 경로 |
| 도보, 운전, 자전거, 대중교통, 항공 등 다양한 형태의 이동 수단을 사용하여 지도에 지정된 두 개 이상의 지점 간의 거리를 표시하고 이동 시간을 보여주는 경로를 그립니다. | Maps Embed API |
| 도보, 운전, 자전거, 대중교통 등 다양한 형태의 이동 수단을 사용한 출발지에서 목적지까지의 경로 데이터를 가져옵니다. | Maps JavaScript API의 경로 서비스(예) Routes API - Compute Routes (웹 서비스) Directions API (기존) (웹 서비스) |
| 지도에서 지형지물과 임의의 데이터를 시각화합니다. | Android용 Maps SDK의 GeoJSON 유틸리티 Maps JavaScript API의 데이터 레이어(예) |
| 히트맵을 사용하여 그라데이션 색상을 사용하여 지리적 지점의 데이터 강도를 시각화합니다. 이러한 데이터의 예로는 인구 밀도, 온도, 교통 체증 등이 있습니다. | Android용 Maps SDK 유틸리티 라이브러리의 히트맵 유틸리티 Maps JavaScript API의 히트맵 레이어(예) |
| 지도에서 KML 데이터를 시각화합니다. | Maps JavaScript API의 KML 레이어(예) Android용 Maps SDK의 KML 유틸리티 |
| 지도에 GeoRSS 데이터를 시각화합니다. | Maps JavaScript API의 KML 레이어(예) |
| 지도에 다각형, 직사각형, 다중선, 원, 마커, 정보 창 (팝업)을 그립니다. | Android용 Maps SDK iOS용 Maps SDK Maps JavaScript API의 오버레이(예) |
| 사용자가 지도에 그릴 수 있는 도구가 포함된 그래픽 인터페이스를 제공합니다. | Maps JavaScript API의 Drawing 라이브러리(예) |
| Google의 3D, 2D, 스트리트 뷰 이미지를 사용하여 렌더링된 생생한 맞춤 지도 시각화를 빌드합니다. | Map Tiles API |
| Google의 3D 지리 공간 이미지를 사용하여 렌더링된 항공 뷰 동영상을 만들고 표시합니다. | Aerial View API |
| 지도에 이미지를 추가합니다(지정된 위치의 오버레이로). | Android용 Maps SDK iOS용 Maps SDK Maps JavaScript API의 지상 오버레이(예) Maps JavaScript API의 맞춤 오버레이(예) |
| 애플리케이션에 대화형 스트리트 뷰 파노라마를 추가하고, 파노라마의 위치를 설정하고, 스트리트 뷰 카메라 (관점)를 조작하는 등의 작업을 할 수 있습니다. | Android용 Maps SDK iOS용 Maps SDK Maps JavaScript API의 스트리트 뷰 서비스(예) Maps Embed API |
| 정적 스트리트 뷰 파노라마를 표시합니다. | Street View Static API |
| 구면기하학 계산을 실행하여 위도와 경도에서 각도, 방향, 거리, 면적을 계산합니다. | Android용 Maps SDK 유틸리티 라이브러리의 구면 기하학 유틸리티 Maps JavaScript API의 Geometry 라이브러리(예) |
| 위도/경도 좌표 시퀀스 (LatLngs)를 인코딩된 경로 문자열로 인코딩하고 인코딩된 경로 문자열을 LatLngs 시퀀스로 디코딩합니다. | Android용 Maps SDK 유틸리티 라이브러리의 폴리라인 유틸리티 Maps JavaScript API의 Geometry 라이브러리 |
| URL을 통해 iOS, Android 또는 웹에서 Google 지도 앱을 엽니다. 앱, 이메일 메시지 또는 웹페이지에서 Google 지도를 실행하는 편리한 방법입니다. 예를 들어 경로 요청과 함께 지도 앱을 실행하여 사용자에게 세부 경로 안내를 제공합니다. | 범용 크로스 플랫폼 지도 URL Android용 Maps SDK의 인텐트 iOS용 Maps SDK의 Google 지도 URL 스키마 |
| 지도 디자인 맞춤설정 | Android용 Maps SDK iOS용 Maps SDK Maps JavaScript API의 스타일(예) Maps Static API |
| 벡터 기반 이미지를 사용하여 SVG 경로 표기법을 사용하여 지도에 있는 마커와 다중선에 사전 정의된 모양과 맞춤 모양을 추가합니다. 이러한 도형의 예로는 화살표, 점선, 맞춤 경로, 경로를 따라 애니메이션 처리된 기호가 있습니다. | Maps JavaScript API의 기호(예) |
| 지도에 대중교통 경로를 표시합니다. | Maps JavaScript API의 대중교통 레이어(예) |
| 지도에 자전거 도로를 표시합니다. | Maps JavaScript API의 자전거 레이어(예) |
| 지도에 현재 교통 상황을 표시합니다. | Maps JavaScript API의 교통 상황 레이어(예) |
| 특정 주소의 위도/경도 좌표를 찾거나 (지오코딩) 지리적 위치의 위도/경도 좌표를 주소로 변환합니다 (역 지오코딩). | Maps JavaScript API의 지오코딩 서비스(예) Geocoding API (웹 서비스) Places API (웹 서비스) |
| 특정 위치의 위성 이미지의 최대 확대/축소 수준을 찾습니다. | Maps JavaScript API의 Google 최대 확대/축소 이미지 서비스(예) |
| 주변 비즈니스 및 기타 장소를 찾거나 장소 유형 또는 쿼리 문자열로 장소를 검색합니다. | Maps JavaScript API의 장소 라이브러리(예) Places API (웹 서비스) |
| 고객 평점 및 리뷰를 포함하여 장소의 이름, 주소, 영업시간 및 기타 세부정보를 가져옵니다. | Maps JavaScript API의 장소 라이브러리(예) Places API (웹 서비스) |
| 특정 지역의 장소 밀도 및 분포를 확인합니다. | Places Aggregate API (웹 서비스) |
| 앱에 Google 지도 검색창의 검색 전 입력 기능 추가 | Android용 Places SDK의 Place Autocomplete iOS용 Places SDK의 Place Autocomplete Maps JavaScript API의 Place Autocomplete 위젯(예) Places API (웹 서비스)의 Place Autocomplete |
| 주소 양식을 자동으로 완성합니다. | Android용 Places SDK의 Place Autocomplete iOS용 Places SDK의 Place Autocomplete Maps JavaScript API의 Place Autocomplete 위젯(예) Places API (웹 서비스)의 Place Autocomplete |
| 주소가 실제 주소인지 평가합니다. | Address Validation |
| 장소에 관한 자세한 정보 제공: 이름과 주소, 위도/경도 좌표, 장소 유형 (나이트클럽, 매장, 박물관) 등 | Android용 Places SDK iOS용 Places SDK |
| 장소 및 Google+ 데이터베이스에서 가져온 비즈니스 및 기타 관심 장소의 사진을 찾습니다. | Android용 Places SDK의 Place Photos Places API (웹 서비스)의 사진 서비스 |
| 주변 비즈니스 및 기타 장소를 포함하여 지도에 표시된 지역의 검색 결과를 표시합니다. | Maps Embed API |
| 모바일 클라이언트가 감지할 수 있는 휴대폰 기지국 및 Wi-Fi 노드에 관한 정보를 토대로 위치 및 정확도 반경을 가져옵니다. 예를 들어, GPS를 사용할 수 없을 때 유용합니다. | Geolocation API (웹 서비스) |
| 기기가 현재 위치한 비즈니스 또는 기타 장소를 가져옵니다. | Android용 Places SDK의 현재 위치 iOS용 Places SDK의 Current Place |
| 지도의 고정 이미지를 표시하고 선택적으로 마커, 다각형, 직사각형, 다중선, 원 및 맞춤 스타일을 추가합니다. 이는 예를 들어 대화형 지도가 필요하지 않고 성능과 크기를 최적화하려는 경우에 유용합니다. | Maps Static API |
| 여러 출발지 및 목적지의 이동 거리와 이동 시간을 계산하며, 선택사항으로 도보, 운전, 자전거 등 다양한 형태의 이동 수단을 지정합니다. | Maps JavaScript API의 Distance Matrix 서비스(예) Routes API - Compute Route Matrix (웹 서비스) Distance Matrix API (기존) (웹 서비스) |
| 해저 위치의 해발 고도와 심도를 확인합니다. | Maps JavaScript API의 고도 서비스(예) Elevation API (웹 서비스) |
| 위치의 시간대 이름, 시간 오프셋 데이터, 일광 절약 시간을 찾습니다. | Time Zone API (웹 서비스) |
| 도로 구간의 속도 제한을 결정합니다. | Roads API (웹 서비스) |
| 차량이 이동한 도로를 확인합니다. GPS 좌표를 차량이 이동한 것으로 보이는 도로에 스냅하거나 가장 가까운 도로 세그먼트를 찾습니다. | Roads API (웹 서비스) |
| 위치에 관한 환경 데이터를 요청합니다. 공기질 및 태양광 데이터를 요청합니다. | Air Quality API (웹 서비스) Solar API (웹 서비스) |
| 특정 위치의 꽃가루 데이터를 요청합니다. Pollen API를 사용하여 65개 이상의 국가에서 특정 위치의 꽃가루 정보(유형, 식물, 지수 포함)를 가져옵니다. | Pollen API (웹 서비스) |
| 특정 위치의 공기질 데이터를 요청합니다. 100개 이상의 국가에서 70개가 넘는 공기질 지수, 오염물질, 건강 권장사항에 대한 데이터를 요청합니다. | Air Quality API (웹 서비스) |
| 특정 위치의 날씨 데이터를 요청합니다. 전 세계 여러 지역의 현재 날씨, 시간별 및 일별 일기예보, 시간별 날씨 기록을 요청할 수 있습니다. | Weather API (웹 서비스) |
| 3억 5천만 개가 넘는 건물의 태양광 발전 가능성에 관한 정보를 요청하세요. Google의 광범위한 매핑 및 컴퓨팅 리소스를 기반으로 하는 자세한 옥상 데이터에 액세스하여 재생 가능한 옥상 태양 에너지 잠재력과 절감액을 추정할 수 있습니다. | Solar API (웹 서비스) |
| 지리공간 데이터를 Google Cloud 콘솔에 업로드합니다. 데이터 기반 스타일 지정에 사용할 수 있도록 Google Cloud 콘솔에 지리 공간 데이터를 저장하고 관리합니다. | Maps Datasets API (웹 서비스) |
| 앱에 세부 경로 안내 추가 모바일 앱을 위한 동적 탐색 환경을 빌드하세요. | Android용 Navigation SDK iOS용 Navigation SDK |
