Title: Maps Platform

의견 보내기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Distance Matrix API (기존) 출발지와 목적지 행렬에 대한 이동 거리와 시간을 가져옵니다.

시작하기

Distance Matrix API를 사용하여 빌드를 시작합니다.

explore [https://developers.google.com/maps/documentation/distance-matrix/cloud-setup?hl=ko]

Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/distance-matrix/cloud-setup?hl=ko]

계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.

add_road [https://developers.google.com/maps/documentation/distance-matrix/start?hl=ko]

거리 행렬 요청 [https://developers.google.com/maps/documentation/distance-matrix/start?hl=ko]

출발지와 목적지 행렬에 대한 이동 거리와 소요 시간을 가져옵니다.

code [https://developers.google.com/maps/documentation/distance-matrix/client-library?hl=ko]

클라이언트 라이브러리 [https://developers.google.com/maps/documentation/distance-matrix/client-library?hl=ko]

Java, Python, Go 또는 Node.js 클라이언트 라이브러리를 사용하여 서버에서 Google 지도 서비스를 사용하세요.

code [https://developers.google.com/maps/documentation/distance-matrix/start?hl=ko#openapi-spec]

OpenAPI 사양 [https://developers.google.com/maps/documentation/distance-matrix/start?hl=ko#openapi-spec]

Postman 컬렉션으로도 제공되는 Distance Matrix API의 OpenAPI 사양을 가져옵니다.

기능

Distance Matrix API의 핵심 기능에 대해 알아보세요.

교통정보 확인 [https://developers.google.com/maps/documentation/distance-matrix/distance-matrix?hl=ko#distance-matrix-advanced]

현재 교통상황을 기반으로 이동 시간을 계산하는 거리 행렬 요청을 생성합니다.

도로의 측면 지정 [https://developers.google.com/maps/documentation/distance-matrix/distance-matrix?hl=ko#location-modifiers]

계산된 경로가 도로의 특정 면을 통과해야 하는지 여부를 지정합니다.

앱 예

Maps JavaScript API를 사용하여 로컬 컴퓨터와 즐겨 사용하는 코드 플레이그라운드에서 라이브 코드 샘플을 실행합니다.

code [https://developers.google.com/maps/documentation/javascript/examples/distance-matrix?hl=ko]

거리 행렬 서비스 [https://developers.google.com/maps/documentation/javascript/examples/distance-matrix?hl=ko]

DistanceMatrixService 객체를 사용하여 위치 집합 간의 거리를 가져옵니다.
