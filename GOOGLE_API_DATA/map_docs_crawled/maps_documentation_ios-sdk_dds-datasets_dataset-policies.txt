데이터 세트를 위한 데이터 기반 스타일 지정 정책
이 문서에는 데이터 세트를 위한 데이터 기반 스타일 사용에 대한 요구사항이 나와 있습니다. Maps Datasets API [https://developers.google.com/maps/documentation/datasets/policies?hl=ko] 서비스를 이용하는 경우 관련 요구사항도 확인해야 합니다.
지도 ID 액세스
지도 ID는 별도의 제한이 없으며 지도 ID와 연결된 데이터 세트 또한 비공개 데이터가 아닙니다. 데이터 세트를 위한 데이터 기반 스타일 지정을 사용하여 만든 지도 및 연결된 스타일과 시각화는 지도 ID와 연결되면 공개적으로 제공되며 해당 지도 ID를 사용하는 모든 사용자가 스타일을 지정할 수 있습니다. 데이터 세트의 데이터 기반 스타일 지정용 소스 파일은 공개적으로 제공되지 않지만 공개적으로 제공되지 않기를 원하는 데이터로 데이터 세트를 만들지 마세요.
자세한 내용은 데이터 세트를 사용한 데이터 기반 스타일 지정 [https://developers.google.com/maps/documentation/ios-sdk/dds-datasets/overview?hl=ko]을 참고하세요.
국가 및 지역 경계
Google 지도에서 국가 또는 지역 경계를 허위로 표시하기 위해 데이터 세트를 위한 데이터 기반 스타일 지정을 통해 데이터를 사용하거나 지도의 지형지물 스타일을 지정하지 마세요.
데이터 저작자 표시
Google에 공유하는 데이터에 적용되는 저작자 표시 요구사항을 준수해야 할 책임은 사용자 본인에게 있습니다. 저작자 표시 텍스트는 Google 로고를 가리거나 Google 로고에 방해가 되어서는 안 됩니다. 저작자 표시에 관한 자세한 내용은 저작자 표시 텍스트 추가 [https://developers.google.com/maps/documentation/ios-sdk/dds-datasets/add-dataset-to-map?hl=ko#add_attribution_text]를 참고하세요.
