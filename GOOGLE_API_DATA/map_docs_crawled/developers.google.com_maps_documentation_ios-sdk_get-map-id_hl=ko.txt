지도 ID 개요

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

플랫폼 선택:

Android [https://developers.google.com/maps/documentation/android-sdk/map-ids/mapid-over?hl=ko]

iOS [https://developers.google.com/maps/documentation/ios-sdk/map-ids/mapid-over?hl=ko]

JavaScript [https://developers.google.com/maps/documentation/javascript/map-ids/mapid-over?hl=ko]

웹 서비스 [https://developers.google.com/maps/documentation/maps-static/map-ids/mapid-over?hl=ko]

유료 기능:
지도 ID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]를 추가하여 액세스하는 기능의 경우, Android 및 iOS용 동적 지도 SKU에 대한 요금이 부과되는 지도 로드가 트리거됩니다. 자세한 내용은 Google 지도 결제 [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#dynamic-maps-ess-sku]를 참고하세요.

지도 ID는 Google Cloud에 저장된 Google 지도 스타일 지정 및 구성 설정을 나타내는 고유 식별자입니다. 지도 ID를 사용하여 웹사이트와 애플리케이션에서 기능을 사용 설정하거나 지도를 관리하거나 스타일을 지정합니다. Google Cloud 콘솔 프로젝트의 지도 관리 페이지에서 필요한 각 플랫폼(JavaScript, Android, iOS 또는 정적 지도)의 지도 ID를 만들 수 있습니다.

지도 ID를 만드는 방법은 지도 ID를 만드는 방법 [https://developers.google.com/maps/documentation/javascript/map-ids/get-map-id?hl=ko]을 참고하세요.

지도 ID로 할 수 있는 작업

지도 ID를 사용하여 기능과 스타일을 사용 설정합니다. 다음은 지도 ID를 사용하는 방법의 몇 가지 예입니다. 전체 목록은 지도 ID를 사용하는 기능 [https://developers.google.com/maps/documentation/javascript/map-ids/mapid-over?hl=ko#features-available]을 참고하세요.

클라우드 기반 지도 스타일 지정: 지도 ID를 지도 스타일과 연결하여 Google Cloud 콘솔을 사용하여 지도의 스타일을 지정하고, 지도를 맞춤설정하고, 지도를 관리합니다.
모든 플랫폼(Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization?hl=ko], iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization?hl=ko], JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization?hl=ko], Maps Static API [https://developers.google.com/maps/documentation/maps-static/cloud-customization?hl=ko])에서 사용할 수 있습니다.
벡터 지도: 지도 ID를 사용하여 WebGL을 사용하여 클라이언트 측에서 로드 시 그려지는 벡터 기반 타일로 구성된 지도를 사용합니다.
JavaScript [https://developers.google.com/maps/documentation/javascript/vector-map?hl=ko]에서 사용할 수 있습니다.
고급 마커: 지도 ID를 사용하여 고급 마커를 사용 설정합니다. Android [https://developers.google.com/maps/documentation/android-sdk/advanced-markers/overview?hl=ko], iOS [https://developers.google.com/maps/documentation/ios-sdk/advanced-markers/overview?hl=ko], JavaScript [https://developers.google.com/maps/documentation/javascript/advanced-markers/overview?hl=ko]에서 사용할 수 있습니다.

클라우드 기반 지도 스타일 지정 예시

클라우드 기반 지도 스타일 지정을 사용하여 웹사이트와 Android 앱의 지도를 스타일 지정하려면 다음 단계를 따르세요.

사용 중인 각 플랫폼의 지도 ID를 만듭니다. 예를 들어 JavaScript 및 Android 지도 ID를 만듭니다. 자세한 내용은 지도 ID 만들기 [https://developers.google.com/maps/documentation/javascript/map-ids/get-map-id?hl=ko]를 참고하세요.
Google Cloud 콘솔에서 지도 스타일을 구성합니다. 자세한 내용은 클라우드 기반 지도 스타일 지정 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko]을 참고하세요.
Google Cloud 콘솔에서 두 지도 ID를 모두 지도 스타일과 연결합니다.
자세한 내용은 스타일에 지도 ID 연결 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko#associate-style-with-map-id]을 참고하세요.
웹사이트 JavaScript 및 Android 앱 코드에서 지도 ID를 참조합니다.
자세한 내용은 앱에 지도 ID 추가하기 [https://developers.google.com/maps/documentation/javascript/map-ids/get-map-id?hl=ko#add-a-map-id-to-your-app]를 참고하세요.

그러면 지도 ID와 연결된 지도 스타일이 웹사이트와 Android 앱에 표시됩니다. Cloud 콘솔에서 지도 스타일을 업데이트하면 고객이 앱을 업데이트하지 않아도 두 위치에 변경사항이 자동으로 표시됩니다.

지도 ID를 사용하는 기능

다음 표에는 지도 ID를 사용하는 Google Maps Platform 기능과 API가 나와 있습니다.

| 기능 또는 API | 지도 ID를 사용하여 이러한 목표를 달성합니다. |
| --- | --- |
| 고급 마커 | 고급 마커를 사용 설정합니다. 지도 ID를 만들 필요가 없으며 대신 데모 지도 ID DEMO_MAP_ID를 사용할 수 있습니다. |
| 경계를 위한 데이터 기반 스타일 지정 | 지도 ID를 경계 및 스타일 지정과 연결하여 경계에 따라 지도의 스타일을 지정합니다. |
| 데이터 세트를 위한 데이터 기반 스타일 지정 | 지도 ID를 데이터 및 스타일 세트와 연결하여 데이터 세트에 따라 지도의 스타일을 지정합니다. |
| Flutter | Flutter 앱에 사용되는 Google 지도의 스타일을 지정합니다. |
| Maps Embed API | 웹페이지에 삽입할 지도를 지정하고 스타일을 지정합니다. |
| Maps JavaScript API | 웹페이지에 표시할 지도의 스타일을 지정합니다. |
| Android용 Maps SDK | Android 애플리케이션에 표시할 지도의 스타일을 지정합니다.1 |
| iOS용 Maps SDK | iOS 애플리케이션에 표시할 지도의 스타일을 지정합니다.1 |
| Maps Static API | 정적 이미지로 렌더링할 지도를 지정하고 스타일을 지정합니다. |
| 이동성 솔루션 | Maps JavaScript API와 Android 및 iOS용 SDK를 사용하여 모빌리티 솔루션에서 지도의 스타일을 지정합니다.1 |
| WebGL (벡터 지도) | JavaScript 벡터 지도 ID를 사용하여 WebGL 기능을 사용 설정합니다. |

1 Android용 Maps SDK 또는 iOS용 Maps SDK에서 지도 ID를 사용하면 Dynamic Maps SKU에 따라 청구되는 지도 로드가 트리거됩니다.
