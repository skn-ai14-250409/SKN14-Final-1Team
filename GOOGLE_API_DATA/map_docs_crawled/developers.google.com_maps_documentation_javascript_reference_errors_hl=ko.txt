MapsNetworkError [https://developers.google.com/maps/documentation/javascript/reference/errors?hl=ko#MapsNetworkError] 클래스

google.maps.MapsNetworkError 클래스

지도에서 네트워크 오류를 관리하기 위한 기본 클래스입니다.
이 클래스는 Error를 확장합니다.

const {MapsNetworkError} = await google.maps.importLibrary("core")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 속성 |
| --- |
| code | 유형:  DirectionsStatus\|DistanceMatrixStatus\|ElevationStatus\|GeocoderStatus\|MaxZoomStatus\|PlacesServiceStatus\|StreetViewStatus API에서 생성된 오류 유형을 식별합니다. |
| endpoint | 유형:  MapsNetworkErrorEndpoint 오류로 응답한 네트워크 서비스를 나타냅니다. |
