HeatmapLayer [https://developers.google.com/maps/documentation/javascript/reference/visualization?hl=ko#HeatmapLayer] 클래스

google.maps.visualization.HeatmapLayer 클래스

지리적 지점의 데이터 강도를 나타내는 클라이언트 측 렌더링 히트맵을 제공하는 레이어입니다.
이 클래스는 MVCObject [https://developers.google.com/maps/documentation/javascript/reference/event?hl=ko#MVCObject]를 확장합니다.

const {HeatmapLayer} = await google.maps.importLibrary("visualization")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 생성자 |
| --- |
| HeatmapLayer | HeatmapLayer([opts]) 매개변수:  opts:  HeatmapLayerOptions optional HeatmapLayer의 새 인스턴스를 만듭니다. |
| 메서드 |
| --- |
| getData | getData() 매개변수: 없음 반환 값:  MVCArray<LatLng\|WeightedLocation> 이 히트맵에 현재 표시되는 데이터 포인트를 반환합니다. |
| getMap | getMap() 매개변수: 없음 반환 값:  Map\|undefined |
| setData | setData(data) 매개변수:  data:  MVCArray<LatLng\|WeightedLocation>\|Array<LatLng\|WeightedLocation> 반환 값:  없음 이 히트맵에 표시할 데이터 포인트를 설정합니다. |
| setMap | setMap(map) 매개변수:  map:  Map optional 반환 값:  없음 지정된 지도에 히트맵을 렌더링합니다. 지도가 null로 설정되면 히트맵이 삭제됩니다. |
| setOptions | setOptions(options) 매개변수:  options:  HeatmapLayerOptions optional 반환 값:  없음 |
| 상속: addListener, bindTo, get, notify, set, setValues, unbind, unbindAll |

HeatmapLayerOptions [https://developers.google.com/maps/documentation/javascript/reference/visualization?hl=ko#HeatmapLayerOptions] 인터페이스

google.maps.visualization.HeatmapLayerOptions 인터페이스

이 객체는 HeatmapLayer 객체에 설정할 수 있는 속성을 정의합니다.
| 속성 |
| --- |
| data optional | 유형:  MVCArray<LatLng\|WeightedLocation>\|Array<LatLng\|WeightedLocation> optional 표시할 데이터 포인트입니다. 필수 항목입니다. |
| dissipating optional | 유형:  boolean optional 확대 시 열지도가 사라지는지 여부를 지정합니다. 기본적으로 데이터 포인트의 영향 범위는 반경 옵션으로만 지정됩니다. 소멸이 사용 중지되면 반경 옵션은 확대/축소 수준 0의 반경으로 해석됩니다. |
| gradient optional | 유형:  Array<string> optional 열지도의 색상 그라데이션으로, CSS 색상 문자열 배열로 지정됩니다. 확장되었으며 이름이 지정된 색상을 제외하고 모든 CSS3 색상이 지원됩니다. |
| map optional | 유형:  Map optional 레이어를 표시할 지도입니다. |
| maxIntensity optional | 유형:  number optional 열지도의 최대 강도입니다. 기본적으로, 열지도 색상은 지도에서 특정 픽셀에 가장 많이 집중된 지점에 따라 동적으로 배율이 조절됩니다. 이 속성을 사용하면 고정된 최대값을 지정할 수 있습니다. |
| opacity optional | 유형:  number optional 기본값: 0.6 열지도의 불투명도로, 0~1 사이의 숫자로 표현됩니다. |
| radius optional | 유형:  number optional 각 데이터 지점의 영향력 반경을 픽셀 단위로 나타냅니다. |

WeightedLocation [https://developers.google.com/maps/documentation/javascript/reference/visualization?hl=ko#WeightedLocation] 인터페이스

| --- |
| location | 유형:  LatLng 데이터 포인트의 위치입니다. |
