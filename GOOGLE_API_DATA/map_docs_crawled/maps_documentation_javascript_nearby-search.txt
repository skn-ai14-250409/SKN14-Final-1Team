주변 지역 검색
플랫폼 선택: Android [https://developers.google.com/maps/documentation/places/android-sdk/nearby-search?hl=ko] iOS [https://developers.google.com/maps/documentation/places/ios-sdk/nearby-search?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/nearby-search?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/places/web-service/nearby-search?hl=ko]
유럽 경제 지역 (EEA) 개발자
주변 지역 검색 (신규)은 하나 이상의 장소 유형을 취하여 일치하는 장소의 목록을 반환합니다. 다음 예에서는 장소 유형을 선택하고 지도에서 결과를 확인할 수 있습니다. 마커를 클릭하면 장소에 대한 자세한 내용을 확인할 수 있습니다.
주변 지역 검색(신규)은 지정한 장소 유형(예: restaurant, book_store, bowling_alley)을 기반으로 일련의 장소에 대한 정보를 반환합니다. 이 서비스는 지정된 locationRestriction의 반경 내에서 지정된 장소 유형과 일치하는 장소의 목록을 반환합니다.
주변 지역 검색(신규)을 사용하려면 Google Cloud 프로젝트에서 'Places API(신규)'를 사용 설정해야 합니다. 자세한 내용은 시작하기 [https://developers.google.com/maps/documentation/javascript/place-get-started?hl=ko]를 참고하세요.
주변 장소 찾기
지정된 장소 유형, 위치, 반경을 기반으로 장소 목록을 반환하려면 searchNearby() [https://developers.google.com/maps/documentation/javascript/reference/place?hl=ko#Place.searchNearby]를 호출합니다. 요청을 사용하여 검색 매개변수를 지정한 다음 searchNearby()를 호출하세요. 결과는 Place 객체의 목록으로 반환되며, 이 목록에서 장소 세부정보 [https://developers.google.com/maps/documentation/javascript/place-details?hl=ko]를 얻을 수 있습니다. 다음 스니펫은 요청 및 searchNearby() 호출의 예를 보여줍니다.
fields 매개변수(필수)를 사용하여 하나 이상의 데이터 필드 [https://developers.google.com/maps/documentation/javascript/place-class-data-fields?hl=ko]가 쉼표로 구분된 목록을 지정합니다.
locationRestriction 매개변수(필수)를 사용하여 최대 50,000미터의 반경을 지정합니다.
includedPrimaryTypes 매개변수를 사용하여 검색할 하나 이상의 장소 유형 [https://developers.google.com/maps/documentation/javascript/place-types?hl=ko]을 지정합니다.
rankPreference 매개변수를 사용하여 POPULARITY 또는 DISTANCE의 SearchNearbyRankPreference를 지정합니다.
매개변수의 전체 목록을 확인합니다. [https://developers.google.com/maps/documentation/javascript/reference/place?hl=ko#SearchNearbyRequest]
