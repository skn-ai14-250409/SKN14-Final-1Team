내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

베타 SDK 지원 중단 알림:
SDK의 베타 버전에 대한 지원과 사용이 중단될 예정입니다. 더 이상 새로운 용도로 사용할 수 없습니다. 이 주제에 설명된 기능은 현재 베타 SDK를 통해 액세스할 수
있습니다. SDK의 향후 버전에서 이러한 기능에 대해 유사한 지원을 할 예정입니다.
자세한 내용은 출시 노트 [https://developers.google.com/maps/documentation/android-sdk/releases?hl=ko#2021-08-18]를 참고하세요.
지도 V3 베타 이전 도구 [https://github.com/googlemaps/android-v3-migration]를
사용하면 지원 중단된 베타 버전에서 지원되는 버전으로 이전하는 데 도움을 받을 수 있습니다.

v.3.1.0 베타 - 2020년 6월 15일

새로운 기능

클라우드 기반 스타일 지정 [https://developers.google.com/maps/documentation/android-sdk/v310-beta?hl=ko#cloud-based-styling]에 대한 지원을 추가했습니다.
마커 충돌 처리 [https://developers.google.com/maps/documentation/android-sdk/v310-beta?hl=ko#marker-collision]에 대한 지원을 추가했습니다.
다중 색상, 그라디언트 색상, 질감이 있는 스타일 지정 등
다중선 스타일 지정 옵션 [https://developers.google.com/maps/documentation/android-sdk/v310-beta?hl=ko#polyline-styling]을 추가했습니다.

API 변경사항

Java와 Kotlin 간의 상호 운용성을 개선하기 위해 InfoWindowAdapter에 null 허용 여부 주석을 추가했습니다.

버그 수정

맞춤 Glide 모듈에서 발생하는 앱 비정상 종료 문제를 해결했습니다(문제 132323222 [https://issuetracker.google.com/issues/132323222?hl=ko]).
doubletap-drag 확장에 대해 트리거되지 않는 OnCameraMove 이벤트를 해결했습니다.
맞춤 타일 오버레이가 있는 지도의 기본 지도 로드 시간을 개선했습니다.
Talkback 모드에서 지도 라이트 클릭 리스너를 호출할 수 있습니다.
Talkback 모드에서 마커가 있는 MapView의 비정상 종료 문제를 해결했습니다.

v.3.0.0 베타 - 2019년 5월 7일

이제 Android용 Maps SDK는 Google Play 서비스를 통해 제공되는 것 외에 독립형 정적 라이브러리를 통해서도 배포됩니다. 이전에는 Android용 Maps SDK가 Google Play 서비스의 일환으로만 제공되었습니다.
Android 4.0-4.0.4('Ice Cream Sandwich')는 더 이상 지원되지 않습니다.
지도 렌더링을 현대화하고 개선했습니다.
이제 관심 장소의 텍스트 라벨이 단순히 아이콘이 아니라 클릭 가능하도록 변경되었습니다.
이제 비즈니스 관심 장소가 실내 지도에 표시됩니다.
도로 공사, 교통사고, 교통체증 등의
이슈 보고서 아이콘이 교통정보 레이어에 추가되었으며, 일반, 혼합, 지형 모드에서 표시됩니다.
에펠탑과 같이 전체 모델이 포함된 3D 빌딩에 대한 지원을 추가했습니다.
Google 지도 애플리케이션과 일치하도록 3D 빌딩의 불투명도를 높였습니다.
마커가 느리게 로드되는 문제를 해결했습니다(문제 35822423 [https://issuetracker.google.com/issues/35822423?hl=ko]).
