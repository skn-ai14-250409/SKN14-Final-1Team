Places Android KTX 개요 및 설정

Places Android KTX는 Android용 Places SDK를 위한 Kotlin 확장 프로그램의 모음입니다. 이 확장 프로그램에서는 Android용 Places SDK 기반으로 개발할 때 간결하고 자연스러운 Kotlin을 작성할 수 있도록 하는 Kotlin 언어 기능을 제공합니다.
Places KTX는 오픈소스로 제공되며 GitHub [https://github.com/googlemaps/android-places-ktx]에서 예시와 함께 제공됩니다.
참고: 버전 4.0.0 [https://developers.google.com/maps/documentation/places/android-sdk/release-notes?hl=ko]부터 Android용 Places SDK에는 Places Android KTX 확장 프로그램 대신 사용할 수 있는 네이티브 Kotlin API가 포함되어 있습니다. 자세한 내용은 Android용 Places SDK 참조 개요 [https://developers.google.com/maps/documentation/places/android-sdk/reference?hl=ko]를 참고하세요.
Places SDK용 KTX 설치

Android용 Places SDK를 위한 KTX를 설치하려면 앱 수준 build.gradle.kts 파일에 다음 종속 항목을 추가하세요.

dependencies {

Android 스튜디오에서 파일 -> 열기를 선택하고 디렉터리로 이동하여 방금 복제하거나 다운로드한 폴더를 엽니다.
데모 앱에 API 키를 추가합니다.

Places API 키를 가져옵니다 [https://developers.google.com/maps/documentation/places/android-sdk/get-api-key?hl=ko].
루트 디렉터리에서 secrets.properties라는 파일을 만듭니다. API 키를 보호하기 위해 이 파일의 버전을 제어해서는 안 됩니다.
secrets.properties에 다음 한 줄을 추가합니다.
PLACES_API_KEY="YOUR_API_KEY"
여기서 YOUR_API_KEY는 첫 단계에서 획득한 실제 API 키입니다. local.defaults.properties [https://github.com/googlemaps/android-places-ktx/blob/main/local.defaults.properties]을 예로 살펴볼 수 있습니다.

실행 구성에서 app-places-ktx 모듈을 선택합니다.
'app-places-ktx' 실행을 선택합니다.
