그래픽으로 마커 만들기
샘플 소스 코드 전체 보기
Advanced Markers API를 사용하면 기본 마커 아이콘을 맞춤 그래픽 이미지로 바꿀 수 있습니다. 글리프를 이미지 파일로 바꿀 수도 있습니다. 이 페이지에서는 다음 방법을 보여줍니다.
맞춤 그래픽 파일 사용 [https://developers.google.com/maps/documentation/javascript/advanced-markers/graphic-markers?hl=ko#custom-graphic-file]
인라인 SVG 사용 [https://developers.google.com/maps/documentation/javascript/advanced-markers/graphic-markers?hl=ko#inline-svg]
그래픽 파일을 글리프로 사용 [https://developers.google.com/maps/documentation/javascript/advanced-markers/graphic-markers?hl=ko#graphic-glyph]
장소 아이콘 이미지를 글리프로 사용 [https://developers.google.com/maps/documentation/javascript/advanced-markers/graphic-markers?hl=ko#place-icon-glyph]
아이콘 글꼴 사용 [https://developers.google.com/maps/documentation/javascript/advanced-markers/graphic-markers?hl=ko#icon-fonts]
고급 마커는 PNG, JPEG, GIF, SVG, TIFF 등 모든 일반적인 이미지 파일 형식 [https://developer.mozilla.org/docs/Web/Media/Formats/Image_types#common_image_file_types]을 지원합니다.
지도에 마커를 추가하려면 먼저 AdvancedMarkerElement 및 PinElement 클래스를 제공하는 marker 라이브러리를 로드 [https://developers.google.com/maps/documentation/javascript/advanced-markers/add-marker?hl=ko#load_the_advanced_marker_library]해야 합니다.
맞춤 그래픽 파일 사용
그래픽 파일(예: PNG)을 참조하는 img 요소를 AdvancedMarkerElement.content 옵션에 전달하여 맞춤 이미지를 사용하여 마커를 만들 수 있습니다.
인라인 SVG 사용
인라인 SVG를 사용해야 하는 경우도 있습니다. SVG XML 요소를 AdvancedMarkerElement.content 옵션에 전달하여 인라인 SVG 이미지를 사용하여 마커를 만들 수 있습니다.
그래픽 파일을 글리프로 사용
그래픽 파일 리소스 또는 인라인 SVG 리소스를 참조하는 img 요소를 PinElement.glyph 옵션에 전달하여 기본 글리프를 바꿀 수 있습니다.
장소 아이콘을 글리프로 사용
장소 세부정보를 사용하면 마커에 적용할 수 있는 장소 아이콘과 배경 색상 [https://developers.google.com/maps/documentation/javascript/place-icons?hl=ko]을 요청할 수 있습니다. place.icon_background_color를 PinElement.background 옵션에 전달하고 place.icon_mask_base_uri를 사용하는 URL을 PinElement.glyph에 전달하여 장소 데이터로 마커를 맞춤설정합니다. .png 또는 .svg 파일 형식 확장자를 URI에 추가합니다. place.geometry.location을 사용하여 마커를 올바른 위치에 배치합니다. 또한 이 예에서는 마커 제목에 place.name을 표시합니다.
아이콘 글꼴 사용
아이콘 이름 및 스타일을 HTML <i> 태그에 CSS 클래스로 추가하고 <div> 태그에 추가하여 Font Awesome과 같은 아이콘 글꼴을 마커와 함께 사용할 수 있습니다. 다음 예에서는 아이콘이 글리프를 바꾸는 데 사용됩니다.
이전 예에서는 필수 Font Awesome 스크립트가 HTML 파일의 script 요소를 통해 로드되었습니다.
<script src="https://use.fontawesome.com/releases/v6.2.0/js/all.js"></script>
