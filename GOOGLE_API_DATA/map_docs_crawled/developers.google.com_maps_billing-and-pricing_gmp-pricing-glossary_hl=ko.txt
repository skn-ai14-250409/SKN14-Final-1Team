이 용어집에서는 Google Maps Platform 결제 및 요금 책정에서 사용되는 일반적인 용어를 정의합니다.

청구 가능 이벤트

일반적으로 청구 가능한 이벤트는 통합에서 트리거한 Google Maps Platform 서비스에서 애플리케이션이 데이터를 수신할 때 발생합니다.  대부분의 경우 서비스에서 데이터가 반환될 때 청구 가능한 이벤트가 발생하지만, 경우에 따라 데이터가 반환되는지 여부와 관계없이 특정 종류의 요청으로 청구 가능한 이벤트가 발생합니다.

다음은 청구 가능한 이벤트의 몇 가지 예입니다.

성공적인 지도 로드
Navigation SDK의 대상 요청
2D 지도 타일의 성공적인 반환
요청으로 끝나는 세션
요청의 필드 마스크로 인해 특정 필드가 반환됨

청구 가능한 이벤트의 예는 SKU 세부정보 [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko]를 참고하세요.

참고: 트리거 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#trigger], SKU [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#sku]

통화

요청 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#request]을 참고하세요.

카테고리

다양한 비즈니스 요구사항과 사용 패턴을 충족하는 기능을 기반으로 한 SKU 카테고리입니다. Essentials, Pro, Enterprise 카테고리 [https://developers.google.com/maps/billing-and-pricing/overview?hl=ko#pricing-categories]를 참고하세요.

비용

애플리케이션 또는 웹사이트에서 트리거하는 SKU와 API 사용량의 가격 책정 등급에 따라 API 사용에 대해 청구되는 금액입니다. 가격 책정 등급 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#pricing-tier] 또는 트리거 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#trigger]도 참고하세요.

엔드포인트

AIP 용어집의 API 엔드포인트 서비스 [https://google.aip.dev/9#api-service-endpoint]를 참고하세요. SKU [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#sku]도 참고하세요.

필드 마스크

서비스에서 반환할 필드 목록으로, 요청의 비용과 지연 시간을 간소화합니다.

가격 책정 등급

월별 사용량에 따라 가격이 책정됩니다. 가격 책정 [https://developers.google.com/maps/billing-and-pricing/pricing?hl=ko]을 참조하세요.

제품

AIP 용어집의 API 제품 [https://google.aip.dev/9#api-product]을 참고하세요.
SKU [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#sku]도 참고하세요.

쿼리

요청 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#request]을 참고하세요.

요청

애플리케이션이 필드 마스크 또는 기타 매개변수와 함께 Google 서버에 메서드를 실행하고 Google 시스템이 해당 메시지를 수신하여 데이터 응답을 반환하는 작업입니다. 대답의 크기는 요청의 성격에 따라 달라질 수 있습니다.

SKU

청구 가능한 이벤트에 해당하는 이름입니다. SKU는 청구 명세서에 표시되는 라벨입니다.

청구 가능한 이벤트 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#billable-event] 및 트리거 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#trigger]도 참고하세요.

트리거

청구 가능한 이벤트 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#billable-event]를 생성하는 API 메서드, 클래스, 필드 마스크, 요청 또는 관련 제약 조건입니다.

사용

특정 SKU에 대해 발생한 청구 가능한 이벤트 수입니다. 청구 가능한 이벤트 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#billable-event] 및 SKU [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#sku]도 참고하세요.

사용량 수준

한 달에 청구 가능한 이벤트가 트리거된 횟수로 측정되는 사용량입니다. 청구 가능한 이벤트 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#billable-event] 및 트리거 [https://developers.google.com/maps/billing-and-pricing/gmp-pricing-glossary?hl=ko#trigger]도 참고하세요. 각 등급의 비용은 가격 시트의 기본 가격 [https://developers.google.com/maps/billing-and-pricing/pricing?hl=ko] 및 인도 가격 [https://developers.google.com/maps/billing-and-pricing/pricing-india?hl=ko] 열에 표시됩니다.
