삽입된 JSON 스타일 선언 사용
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/styling?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/styling?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/json-styling-overview?hl=ko]
삽입된 JSON 스타일 선언을 통해 맞춤설정된 스타일을 다음 항목에 적용할 수 있습니다.
기본 지도
사용자가 단일 지도 프레임 내에서 전환할 수 있는 지도 유형
여러 앱에서 동일한 스타일을 업데이트하려면 Google Cloud 콘솔에서 사용할 수 있고 지도 ID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]가 필요한 클라우드 맞춤설정 [https://developers.google.com/maps/documentation/javascript/cloud-customization?hl=ko]을 살펴보세요. 잠재적인 충돌을 방지하려면 동일한 앱에서 클라우드 맞춤설정과 하드코딩된 스타일 지정을 결합하지 마세요.
실험 단계: 더 풍부한 스타일 지정 환경을 위해 클라우드 기반 지도 스타일 지정과 JSON 스타일 지정 [https://developers.google.com/maps/documentation/javascript/cloud-customization/json?hl=ko]을 사용할 수 있습니다. 이 기능에는 더 광범위한 지도 기능 목록과 함께 외관을 맞춤설정할 수 있는 더 많은 옵션이 있습니다.
지형지물에 스타일 지정
지도의 여러 지형지물과 요소에 스타일을 적용하려면 지도의 스타일을 지정하는 방식을 정의하는 MapTypeStyle [https://developers.google.com/maps/documentation/javascript/3.exp/reference?hl=ko#MapTypeStyle] 객체의 배열을 만드세요.
배열은 다음과 같은 형식을 취합니다.
var stylesArray = [
  {
    featureType: '',
    elementType: '',
    stylers: [
      {color: ''},
      {visibility: ''},
      // Add any stylers you need.
    ]
  },
  {
    featureType: '',
    // Add the stylers you need.
  }
]
사용 가능한 모든 featureType, elementType, stylers 값의 목록은 JSON 스타일 참조 [https://developers.google.com/maps/documentation/javascript/style-reference?hl=ko]를 참고하세요.

참고: 여러 앱 플랫폼에 스타일을 적용하려면 클라우드 기반 지도 스타일 지정 [https://developers.google.com/maps/documentation/javascript/cloud-customization?hl=ko]을 사용하는 것이 좋습니다. JSON 스타일을 사용하려면 가져오면 됩니다. 자세한 내용은 JSON 지도 스타일 가져오기 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko#json-import]를 참고하세요.
기본 지도에 스타일 적용
다음 예에서는 어두운 배경(어두운 모드)에 밝은 색상의 텍스트와 아이콘을 사용하여 스타일이 지정된 기본 지도의 예를 보여줍니다. 코드 샘플을 보려면 스타일 지도 - 야간 모드 코드 샘플 [https://developers.google.com/maps/documentation/javascript/examples/style-array?hl=ko]로 이동하세요.
기본 지도의 스타일을 수정하려면 다음과 같은 시점에 MapOptions 객체에서 지도의 styles 속성을 스타일 배열로 설정하세요.

라벨과 도로를 변경하면 지형, 위성, 하이브리드, 기본 로드맵 유형 등 모든 지도 유형이 영향을 받습니다.
지도 유형에 스타일 적용
다음 예에서는 지도의 왼쪽 상단에 있는 컨트롤에 추가된 스타일 지도라는 지도 유형을 보여줍니다. 코드 샘플을 보려면 스타일 지도 유형 [https://developers.google.com/maps/documentation/javascript/examples/maptype-styled-simple?hl=ko]으로 이동하세요.
스타일 지도 유형을 만들려면 스타일 배열을 StyledMapType [https://developers.google.com/maps/documentation/javascript/3.exp/reference?hl=ko#StyledMapType] 객체로 설정하세요. 새 스타일 지도 유형을 만들어도 기본 지도 유형의 스타일은 영향을 받지 않습니다.
