People API에 액세스하는 다른 방법 [https://developers.google.com/people/v1/libraries?hl=ko#other]
People API는 HTTP 및 JSON에 기반하므로 모든 표준 HTTP 클라이언트는 요청을 보내고 응답을 파싱할 수 있습니다.
그러나 Google API 클라이언트 라이브러리는 HTTP를 통한 API 요청에 대한 통합 및 향상된 보안 클라이언트 라이브러리는 다양한 프로그래밍 언어 사전 정의된 임곗값을 사용하면 응답을 파싱합니다
시작하려면 개발에 사용할 프로그래밍 언어를 선택하세요.

이 페이지에는 다음을 사용하여 People API를 시작하는 방법에 대한 정보가 포함되어 있습니다.
Python용 Google API 클라이언트 라이브러리입니다.  자세한 내용은 다음 설명서를 참고하세요.


  People API의 PyDoc 참조 [https://googleapis.github.io/google-api-python-client/docs/dyn/people_v1.html] 찾아보기
  Python용 Google API 클라이언트 라이브러리 개발자 가이드 [https://developers.google.com/api-client-library/python?hl=ko] 읽기
  People API용 API 탐색기 [https://developers.google.com/apis-explorer/?hl=ko#p/people/v1/]를 사용하여 브라우저에서 이 API와 상호작용






  시스템 요구사항
  
    운영체제:
      
        Linux
        macOS X
        Windows
      
    
    Python 2.7 또는 3.4 이상 [http://python.org/download/]
  


  클라이언트 라이브러리 설치하기
  패키지 관리자를 사용하거나 Python 클라이언트 라이브러리를 직접 다운로드하여 설치할 수 있습니다.
  관리형 설치
  pip 또는 setuptools를 사용하여 설치를 관리합니다(먼저 sudo를 실행해야 할 수 있음).
  
    pip [http://pypi.python.org/pypi/pip](선호됨):
      pip install --upgrade google-api-python-client
    
    Setuptools [http://pypi.python.org/pypi/setuptools]:
      easy_install --upgrade google-api-python-client
    
  
    직접 설치
  
    Python용 최신 클라이언트 라이브러리를 다운로드합니다 [https://pypi.python.org/pypi/google-api-python-client/].
    코드를 압축해제합니다.
    설치합니다.
   python setup.py install
  
  App Engine
  Python 클라이언트 라이브러리는 App Engine Python 런타임 환경 [https://cloud.google.com/appengine/docs/python/?hl=ko]에 설치되어 있지 않으므로 타사 라이브러리와 마찬가지로 애플리케이션에 복사 [https://cloud.google.com/appengine/docs/python/tools/libraries27?hl=ko#vendoring]해야 합니다.

  google-api-client gem 설치
    
    시스템에 따라 명령어 앞에 sudo를 추가해야 할 수 있습니다.
  
  
    Ruby용 Google API 클라이언트 라이브러리를 설치하지 않았으면 RubyGems를 사용하여 설치합니다.
     gem install google-api-client
  
  이미 gem을 설치했다면 최신 버전으로 업데이트합니다.
     gem update -y google-api-client
