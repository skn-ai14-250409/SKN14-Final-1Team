Atom 정보
Atom은 뉴스 웹사이트, 블로그 또는 Gmail의 정기 업데이트를 한곳에서 쉽게 받을 수 있는 시스템입니다. 애그리게이터 (뉴스 리더, 피드 리더 또는 RSS/Atom 리더라고도 함)와 함께 Atom을 사용하여 새 메시지 알림을 받을 수 있습니다.
참고: 이 피드는Google Workspace 도메인
의 Gmail 계정에서만 사용할 수 있습니다.
받은편지함 피드 사용하기
Gmail 받은편지함 피드는 받은편지함을 XML 문서로 출력합니다. RSS 애그리게이터로 이를 확인하거나 자체 앱으로 피드를 사용할 수 있습니다. OAuth 2.0 [https://developers.google.com/accounts/docs/OAuth2?hl=ko]이 권장되는 인증 방법으로, 범위 https://mail.google.com/mail/feed/atom를 사용하고 요청 GET https://mail.google.com/mail/feed/atom로 피드를 가져옵니다.
