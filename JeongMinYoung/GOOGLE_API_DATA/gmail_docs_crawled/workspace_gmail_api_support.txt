Google은 다양한 플랫폼을 함께 사용하여 개발자에게 지원을 제공하고 있습니다. 아래에서 내게 가장 적합한 지원 옵션이 무엇인지 알아보세요.

Stack Overflow
Google은 널리 사용되는 프로그래밍 Q&A 웹사이트인 Stack Overflow [https://stackoverflow.com/questions/tagged/gmail-api]에서 기술적인 질문을 해결해 드립니다. Google은 이 사이트를 소유하거나 관리하지 않지만 Google 계정으로 로그인할 수 있습니다.
Stack Overflow는 다양한 주제에 대한 질문을 포괄하며, 개발자는 이 서비스와 관련된 질문을 [gmail-api] 태그로 표시합니다. 질문에 추가 태그를 포함하여 관련 기술 전문가의 주의를 끌 수 있습니다.
기존 질문 검색 새 질문하기 [https://stackoverflow.com/questions/ask?tags=gmail-api,google-workspace]
개발자 제품 의견
개발자 제품 기능에 관한 의견이 있는 경우 문제 추적기를 검색 [https://issuetracker.google.com/issues?q=status%3Aopen+componentid%3A191602%2B+type%3A%28bug%7Cfeature_request%7Ccustomer_issue%29&hl=ko]하여 다른 사용자가 이미 동일한 의견을 제출했는지 확인하세요. 기존 의견 보고서를 찾은 경우 문제 번호 옆에 있는 별표를 클릭하여 동의를 표시하고 가장 중요한 보고서의 우선순위를 지정하는 데 도움을 주세요. 추가적인 맥락이나 정보를 제공하고 싶다면 댓글을 추가하세요.
다른 사용자가 비슷한 의견을 제출하지 않은 경우 새 의견을 제출할 수 있습니다. 중요하다고 생각하는 이유를 포함하여 의견을 최대한 구체적으로 설명해 주세요.
기존 의견 검색 버그 제출 [https://issuetracker.google.com/issues/new?component=191602&template=823912&hl=ko] 기능 요청 제출 [https://issuetracker.google.com/issues/new?component=191602&template=824107&hl=ko]
