다음 라이브러리와 샘플을 사용하면 OAuth 2.0 승인을 사용하여 Gmail IMAP 및 SMTP 서비스에 쉽게 액세스할 수 있습니다.
자바 샘플
JavaMail 1.5.2 이상은 IMAP용 OAuth를 기본적으로 지원 [https://javaee.github.io/javamail/OAuth2]합니다. 이전 버전을 사용해야 하거나 자체 구현을 만들어야 하는 경우 Java 샘플 코드 [https://github.com/google/gmail-oauth2-tools/wiki/JavaSampleCode]를 참고하세요. 샘플을 다운로드하려면 OAuth2로 Gmail에 인증하기 위한 도구 및 샘플 코드 [https://github.com/google/gmail-oauth2-tools]를 참고하세요.
