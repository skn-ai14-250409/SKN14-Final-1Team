원활한 사용자 인증 정보 공유
원활한 사용자 인증 정보 공유를 사용하여 웹사이트와 앱에서 사용자 인증 정보를 공유합니다. 사용자가 웹과 앱에서 로그인 및 가입 상태를 유지하여 불필요한 비밀번호 재설정 및 헬프데스크 호출을 피하세요.
[https://developers.google.com/identity/credential-sharing/fundamentals?hl=ko]
원활한 사용자 인증 정보 공유란 무엇인가요? [https://developers.google.com/identity/credential-sharing/fundamentals?hl=ko]
원활한 사용자 인증 정보 공유는 Google 비밀번호 관리자 및 Google 생태계의 기타 서비스에서 웹사이트와 Android 앱 간에 비밀번호를 안전하게 공유할 수 있도록 하는 일련의 관행, 기술, 인프라입니다.
웹 및 Android 개발자는 사용자 인증 정보 공유를 구현하여 사용자의 로그인 환경을 간소화할 수 있습니다. 이렇게 하면 Google 비밀번호 관리자가 동일한 계정 관리 백엔드를 공유하는 도메인 또는 Android 앱에서 사용자 인증 정보를 자동 완성할 수 있습니다.
예를 들어 사용자가 Chrome을 사용하는 동안 비밀번호 관리자에 도메인의 사용자 인증 정보를 저장하면 Chrome에서 연결된 다른 도메인의 사용자 인증 정보 자동 완성을 제안하거나 Android가 연결된 Android 앱에 자동 완성을 제안할 수 있습니다.
