이 가이드 시리즈에서는 YouTube Data API (v3)를 사용하여 다양한 유형의 함수를 수행하는 방법을 설명합니다. 여기에는 설명, 코드 샘플에 대한 포인터, API 탐색기를 샘플 쿼리로 채우는 링크가 포함됩니다.
이 가이드를 읽을 때는 다음 사항에 유의하세요.
가능한 경우 샘플 v3 API 쿼리는 쿼리에 표시된 값으로 API 탐색기를 연결하고 채웁니다. 이 예시를 통해 간편하게 테스트 쿼리를 제출하고 샘플 응답을 확인할 수 있습니다.
일부 샘플 요청에는 승인이 필요합니다. API 탐색기 양식 상단에는 OAuth 2.0 승인을 사용 설정하는 작은 위젯이 있습니다.
애플리케이션에서 YouTube Data API를 사용하려면 먼저 승인 사용자 인증 정보를 획득 [https://developers.google.com/youtube/registering_an_application?hl=ko]해야 합니다.
