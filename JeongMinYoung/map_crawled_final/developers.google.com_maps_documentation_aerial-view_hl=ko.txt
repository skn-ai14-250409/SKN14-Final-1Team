의견 보내기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Aerial View API Google의 3D 지리 공간 이미지를 사용하여 렌더링된 항공 뷰 동영상을 만들고 표시할 수 있습니다. 항공 뷰 동영상은 상공에서 선회하는 드론이 촬영한 동영상을 시뮬레이션하여 관심 장소의 실사 3D 항공 뷰입니다.

시작하기

Aerial View API를 사용하여 빌드를 시작합니다.

explore [https://developers.google.com/maps/documentation/aerial-view/cloud-setup?hl=ko]

Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/aerial-view/cloud-setup?hl=ko]

계정을 만들고, API 키를 생성하고, 세션 토큰을 생성한 다음, 빌드를 시작하세요.

travel_explore [https://developers.google.com/maps/documentation/aerial-view/lookup-video?hl=ko]

기존 항공 뷰 동영상 요청 [https://developers.google.com/maps/documentation/aerial-view/lookup-video?hl=ko]

이 사용법에서는 필요한 모든 항공 뷰 동영상을 이미 Google에서 생성했다고 가정합니다.

description [https://developers.google.com/maps/documentation/aerial-view/lookup-video-metadata?hl=ko]

동영상 메타데이터 요청 [https://developers.google.com/maps/documentation/aerial-view/lookup-video-metadata?hl=ko]

최신 상태 및 길이 등 특정 동영상과 관련된 메타데이터를 봅니다.

travel_explore [https://developers.google.com/maps/documentation/aerial-view/generate-video?hl=ko]

새 항공 뷰 동영상 생성 [https://developers.google.com/maps/documentation/aerial-view/generate-video?hl=ko]

이 사용법에서는 필요한 항공 뷰 동영상이 Google에서 아직 생성되지 않았으므로 새 항공 사진을 생성해야 한다고 가정합니다.

code [https://developers.google.com/maps/documentation/aerial-view/reference/rest?hl=ko]

API 참조 보기 [https://developers.google.com/maps/documentation/aerial-view/reference/rest?hl=ko]

Aerial View REST 및 gRPC API 참조를 확인합니다.
