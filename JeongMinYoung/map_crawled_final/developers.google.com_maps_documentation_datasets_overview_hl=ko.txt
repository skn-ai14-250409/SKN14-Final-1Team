Maps Datasets API 개요

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

데이터 세트는 데이터 기반 스타일 지정의 일부로 Google Maps Platform 앱에서 사용할 데이터의 컨테이너입니다.
Maps Datasets API를 사용하면 REST API를 사용하여 데이터 세트를 만들고 관리할 수 있습니다.
참고: Maps Datasets API를 사용하는 경우 요금이 부과되지 않습니다.
예를 들어 데이터 세트에 대한 데이터 기반 스타일 지정을 사용하면 자체 지리 정보 데이터를 데이터 세트에 업로드하고, 데이터 지형지물에 맞춤 스타일을 적용하고, 지도에 이러한 데이터 지형지물을 표시할 수 있습니다. 점, 다중선, 다각형 도형을 기반으로 데이터 시각화를 생성하고 데이터 지형지물이 클릭 이벤트에 반응하도록 설정할 수 있습니다.

데이터 세트 만들기는 두 단계로 이루어집니다.

데이터 세트를 만드는 요청을 합니다.
데스크톱 또는 Google Cloud Storage에서 데이터 세트로 데이터를 업로드하도록 요청합니다. 업로드 데이터는 CSV, GeoJSON 또는 KML 파일로 표시되어야 합니다.

데이터 기반 스타일 지정의 일부로 데이터 세트를 사용하는 방법에 대한 자세한 내용은 다음을 참고하세요.

Maps JavaScript API [https://developers.google.com/maps/documentation/javascript/dds-datasets/overview?hl=ko].
Android용 Maps SDK [https://developers.google.com/maps/documentation/android-sdk/dds-datasets/overview?hl=ko]
iOS용 Maps SDK [https://developers.google.com/maps/documentation/ios-sdk/dds-datasets/overview?hl=ko]

Maps Datasets API 작동 방식

Maps Datasets API에는 다음과 같은 메서드가 있습니다.

데이터 세트 만들기: create [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/create?hl=ko] 엔드포인트를 사용하여 데이터 세트를 만듭니다.
데이터 업로드: upload [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/media/upload?hl=ko] 엔드포인트를 사용하여 데이터 세트에 데이터를 업로드합니다.
데이터 세트 나열: list [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/list?hl=ko] 엔드포인트를 사용하여 모든 데이터 세트 목록을 가져옵니다.
데이터 세트 가져오기: get [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/get?hl=ko] 엔드포인트를 사용하여 특정 데이터 세트에 관한 정보를 가져옵니다.
데이터 세트 오류 가져오기: fetchDatasetErrors [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/fetchDatasetErrors?hl=ko] 엔드포인트를 사용하여 데이터 세트에 관한 오류 정보를 가져옵니다.
데이터 세트 업데이트: patch [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/patch?hl=ko] 엔드포인트를 사용하여 특정 데이터 세트에 관한 정보를 업데이트합니다.
데이터 세트 다운로드: download [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/media/download?hl=ko] 엔드포인트를 사용하여 데이터 세트에서 데이터를 다운로드합니다.
데이터 세트 삭제: delete [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets/delete?hl=ko] 엔드포인트를 사용하여 데이터 세트를 삭제합니다.
