시작하기

Street View Static API를 사용해 지도 작성을 시작합니다.

explore [https://developers.google.com/maps/documentation/streetview/cloud-setup?hl=ko]

Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/streetview/cloud-setup?hl=ko]

계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.

panorama_wide_angle [https://developers.google.com/maps/documentation/streetview/overview?hl=ko]

정적 스트리트 뷰 파노라마 삽입 [https://developers.google.com/maps/documentation/streetview/overview?hl=ko]

<img> 태그를 사용하여 지도에 정적 스트리트 뷰 파노라마를 추가합니다.

panorama_photosphere [https://developers.google.com/maps/documentation/javascript/streetview?hl=ko]

대화형 스트리트 뷰 파노라마 삽입 [https://developers.google.com/maps/documentation/javascript/streetview?hl=ko]

Maps JavaScript API 스트리트 뷰 서비스를 사용하여 대화형 스트리트 뷰 파노라마를 지도에 추가합니다.

add_photo_alternate [https://developers.google.com/maps/documentation/javascript/streetview?hl=ko#CreatingPanoramas]

맞춤 스트리트 뷰 파노라마 제공 [https://developers.google.com/maps/documentation/javascript/streetview?hl=ko#CreatingPanoramas]

자신만의 모습을 담은 맞춤 파노라마를 만들어 보세요.

스트리트 뷰 기능 및 API

스트리트 뷰 이미지를 Android, iOS 및 웹에서 가져와 나만의 구형 이미지를 만들어 360도 파노라마 사진을 Google 지도에 게시하세요.

Android용 Maps SDK의 스트리트 뷰 [https://developers.google.com/maps/documentation/android-sdk/streetview?hl=ko]

Android 기기에서 스트리트 뷰 서비스를 이용할 수 있습니다.

iOS용 Maps SDK의 스트리트 뷰 [https://developers.google.com/maps/documentation/ios-sdk/streetview?hl=ko]

iOS 기기에서 스트리트 뷰 서비스를 이용하세요.

Maps Embed API의 스트리트 뷰 [https://developers.google.com/maps/documentation/embed/get-started?hl=ko]

웹페이지에 스트리트 뷰 이미지를 삽입합니다.

나만의 스트리트 뷰 게시하기 [https://developers.google.com/streetview/publish?hl=ko]

Google 지도에 360도 파노라마 사진을 게시합니다.

구체 카메라(Spherical Camera)와 앱 연결 [https://developers.google.com/streetview/open-spherical-camera?hl=ko]

Open Spherical Camera API 사양을 사용하여 함께 작동하는 구형 카메라와 애플리케이션을 만듭니다.

Photo Sphere 만들기 및 렌더링 [https://developers.google.com/streetview/spherical-metadata?hl=ko]

공개된 Photo Sphere XMP 메타데이터를 기존의 Photo Sphere 및 파노라마 이미지에 삽입합니다.
