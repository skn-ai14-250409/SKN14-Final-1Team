시작하기

iOS용 Maps SDK의 일반적인 기능을 사용하여 지도 작성을 시작합니다.

explore [https://developers.google.com/maps/documentation/ios-sdk/cloud-setup?hl=ko]

Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/ios-sdk/cloud-setup?hl=ko]

계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.

map [https://developers.google.com/maps/documentation/ios-sdk/map-with-marker?hl=ko]

마커가 포함된 첫 지도 만들기 [https://developers.google.com/maps/documentation/ios-sdk/map-with-marker?hl=ko]

iOS용 Maps SDK를 로드하고 마커가 있는 지도를 웹 앱에 추가하는 방법을 알아봅니다.

palette [https://developers.google.com/maps/documentation/ios-sdk/maps-customization-overview?hl=ko]

지도 스타일 맞춤설정 [https://developers.google.com/maps/documentation/ios-sdk/maps-customization-overview?hl=ko]

도로, 지형 지물, 관심 장소 등 지도의 모든 측면을 맞춤설정합니다.

chat [https://developers.google.com/maps/documentation/ios-sdk/current-place-tutorial?hl=ko]

지도에서 현재 장소를 선택하고 세부정보를 표시합니다. [https://developers.google.com/maps/documentation/ios-sdk/current-place-tutorial?hl=ko]

iOS 기기의 현재 위치를 찾고 해당 위치에 있는 장소나 다른 관심 장소의 세부정보를 표시합니다.

특성

iOS용 Maps SDK의 핵심 기능에 대한 문서를 찾아봅니다.

동적 지도 [https://developers.google.com/maps/documentation/ios-sdk/map?hl=ko]

iOS 앱에 기본 지도를 추가합니다.

마커 [https://developers.google.com/maps/documentation/ios-sdk/marker?hl=ko]

지도에 마커를 추가하고, 클릭 이벤트와 상호작용하도록 설정하며, 색상, 이미지 등을 맞춤설정합니다.

컨트롤 및 동작 [https://developers.google.com/maps/documentation/ios-sdk/controls?hl=ko]

기본으로 제공되는 UI 구성요소 및 동작을 구성하여 사용자가 상호작용하는 방식을 맞춤설정합니다.

이벤트 [https://developers.google.com/maps/documentation/ios-sdk/events?hl=ko]

지도 클릭, 마커 클릭, 카메라 변경, 오버레이 이벤트 등을 비롯한 지도상의 이벤트를 수신합니다.

도형 [https://developers.google.com/maps/documentation/ios-sdk/shapes?hl=ko]

지도에 다각형, 다중선, 원을 추가하여 지도 모양을 맞춤설정합니다.

지면 오버레이 [https://developers.google.com/maps/documentation/ios-sdk/overlays?hl=ko]

위도와 경도에 연결된 이미지 오버레이를 추가하고 이를 움직여 지도를 드래그하거나 확대/축소합니다.

카메라 및 뷰 [https://developers.google.com/maps/documentation/ios-sdk/views?hl=ko]

위치, 확대/축소 수준, 방위 등 카메라의 다양한 측면을 설정합니다.

타일 레이어 [https://developers.google.com/maps/documentation/ios-sdk/tiles?hl=ko]

Google의 기본 지도 타일 위에 이미지를 겹쳐서 데이터와 로컬 이미지를 앱에 추가합니다.

스트리트 뷰 [https://developers.google.com/maps/documentation/ios-sdk/streetview?hl=ko]

간단한 HTTP 요청을 사용하여 비대화형 스트리트 뷰 파노라마 또는 미리보기 이미지를 웹페이지에 삽입합니다.
