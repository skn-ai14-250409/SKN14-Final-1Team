Google 지도 도메인 목록

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

다음은 지도의 구성요소가 로드될 때 Google Maps Platform에서 사용되는
모든 도메인의 목록입니다. 인터넷 액세스가 도메인별 정책에 의해
필터링되는 경우 이 목록을 사용하여 방화벽 구성을 설정하세요.

이 목록은 새 기능이 출시되면 변경될 수 있으며 최신 목록이
아닐 수 있습니다. 2개월마다 이 페이지를 확인할 것을 권장합니다.

Maps JavaScript API

Maps JavaScript API 부트스트랩 및 서비스(지오코딩, 운전 경로, 고도, KML, 일부 타일)에서는 다음 호스트 이름을 사용합니다.

maps.googleapis.com

이 도메인에 대한 XHR 호출이 승인되도록 콘텐츠 보안 정책을 업데이트해야 할 수도 있습니다. 자세한 내용은 FAQ [https://developers.google.com/maps/faq?hl=ko#uptime_slo]를 참고하세요.
