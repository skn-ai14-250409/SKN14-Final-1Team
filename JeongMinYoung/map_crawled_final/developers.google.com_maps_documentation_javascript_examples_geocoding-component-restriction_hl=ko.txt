This example creates a request to the geocoding service that restricts the
results to a particular postal area within Sydney, Australia.

Read the
documentation [https://developers.google.com/maps/documentation/javascript/geocoding#ComponentFiltering].

Clone Sample

Git and Node.js are required to run this sample locally. Follow these instructions [https://docs.npmjs.com/downloading-and-installing-node-js-and-npm] to install Node.js and NPM. The following commands clone, install dependencies and start the sample application.

git clone -b sample-geocoding-component-restriction https://github.com/googlemaps/js-samples.git
cd js-samples
npm i
npm start

Other samples can be tried by switching to any branch beginning with sample-SAMPLE_NAME.

git checkout sample-SAMPLE_NAME
npm i
