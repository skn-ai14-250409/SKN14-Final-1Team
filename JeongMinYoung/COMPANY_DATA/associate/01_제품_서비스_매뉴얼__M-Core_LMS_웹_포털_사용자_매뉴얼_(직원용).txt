M-Core LMS 웹 포털 사용자 매뉴얼 (직원용)

1) 개요
M-Core LMS 웹 포털은 직원들이 교육 자료에 접근하고, 학습 진행 상황을 관리하며, 교육 관련 정보를 공유하는 플랫폼입니다. 주요 시나리오는 다음과 같습니다: 
- 교육 자료 검색 및 다운로드
- 교육 이수 현황 확인 및 관리
- 동료와의 협업 및 피드백 제공

2) 접근/권한
- 최소권한 원칙: 모든 직원은 본인의 역할에 맞는 최소한의 권한만 부여받습니다. 불필요한 권한 요청은 지양해 주세요.
- 계정 요청 방법: 신규 계정이 필요할 경우, 인사팀에 이메일로 요청서를 제출하세요. 요청서에는 이름, 부서, 직위, 필요한 권한을 명시해야 합니다.

3) 기본 사용 절차
1. 웹 포털 접속: M-Core LMS 웹 포털 URL 입력 후 로그인 화면으로 이동합니다.
   - 검증 포인트: 로그인 화면이 정상적으로 표시되는지 확인합니다.
2. 사용자 ID 및 비밀번호 입력: 회사에서 부여받은 ID와 비밀번호를 입력합니다.
   - 검증 포인트: 로그인 후 대시보드가 정상적으로 표시되는지 확인합니다.
3. 교육 자료 검색: 대시보드에서 '교육 자료' 메뉴를 클릭하여 원하는 자료를 검색합니다.
   - 검증 포인트: 검색 결과가 정확하게 나타나는지 확인합니다.
4. 교육 이수 현황 확인: '내 학습' 메뉴에서 진행 중인 교육 및 이수 완료 교육을 확인합니다.
   - 검증 포인트: 이수 현황이 정확하게 표시되는지 확인합니다.
5. 피드백 제공: 동료의 교육 자료에 대해 피드백을 제공하려면 해당 자료 페이지에서 '피드백' 버튼을 클릭합니다.
   - 검증 포인트: 피드백이 정상적으로 제출되었는지 확인합니다.

4) 운영 체크리스트
- 일일 체크리스트:
  - 로그인하여 대시보드 확인
  - 새로운 교육 자료 업데이트 확인
  - 피드백 요청 확인 및 응답
- 주간 체크리스트:
  - 이수 현황 검토 및 미이수 교육 확인
  - 동료와의 협업 진행 상황 점검
  - 필요한 교육 자료 요청

5) 문제 해결 FAQ
1. 로그인할 수 없습니다. 어떻게 해야 하나요?
   - 비밀번호를 재설정해 보세요. 재설정 방법은 '비밀번호 찾기' 링크를 클릭하여 안내를 따릅니다.
2. 교육 자료가 검색되지 않습니다. 이유는 무엇인가요?
   - 검색어를 확인하고, 필터를 조정해 보세요. 자료가 업데이트되었는지 확인합니다.
3. 피드백이 제출되지 않았습니다. 무엇이 문제인가요?
   - 인터넷 연결 상태를 확인하고 다시 시도해 보세요. 또한, 모든 필드를 올바르게 입력했는지 확인합니다.
4. 계정이 잠겼습니다. 어떻게 해야 하나요?
   - 인사팀에 문의하여 계정 잠금 해제를 요청하세요.
5. 교육 이수 현황이 업데이트되지 않았습니다. 어떻게 해결하나요?
   - 이수한 교육의 완료 버튼을 클릭했는지 확인하고, 문제가 지속되면 관리자에게 문의합니다.

6) 보안/컴플라이언스
- 비밀번호는 주기적으로 변경하고, 타인과 공유하지 말아야 합니다.
- 외부에서 접근할 경우, VPN을 사용하여 보안을 강화하세요.
- 교육 자료 및 개인 정보를 안전하게 관리하고, 불필요한 정보는 삭제합니다.
