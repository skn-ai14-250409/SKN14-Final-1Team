M-Core 콘텐츠 업로더 & 저작도구 사용 매뉴얼 (직원용)

1) 개요
M-Core의 콘텐츠 업로더 및 저작도구는 다양한 디지털 교육 콘텐츠를 생성하고 관리하는 데 사용됩니다. 주요 시나리오는 다음과 같습니다:
- 콘텐츠 업로드: 새로운 교육 자료를 시스템에 추가하여 사용자에게 제공.
- 콘텐츠 수정: 기존 콘텐츠를 업데이트하거나 수정하여 최신 정보를 반영.
- 콘텐츠 삭제: 더 이상 필요하지 않은 콘텐츠를 안전하게 제거.

2) 접근/권한
- 최소권한 원칙: 모든 직원은 업무에 필요한 최소한의 권한만 보유해야 합니다. 콘텐츠 업로더 및 저작도구에 접근하기 위해서는 해당 권한이 필요합니다.
- 계정 요청 방법: 권한 요청은 인사팀에 이메일로 제출해야 하며, 요청 시 본인의 이름, 부서, 요청 이유를 명시해야 합니다.

3) 기본 사용 절차
1단계: 로그인
- 시스템에 접속하여 개인 계정으로 로그인합니다.
- 검증 포인트: 로그인 후 대시보드가 정상적으로 표시되는지 확인합니다.

2단계: 콘텐츠 업로드
- '콘텐츠 업로드' 버튼을 클릭합니다.
- 파일 선택 창에서 업로드할 파일을 선택한 후, '업로드'를 클릭합니다.
- 검증 포인트: 업로드 완료 메시지가 표시되고, 대시보드에서 새로운 콘텐츠가 나타나는지 확인합니다.

3단계: 콘텐츠 수정
- 수정할 콘텐츠를 선택하고 '수정' 버튼을 클릭합니다.
- 필요한 내용을 수정한 후, '저장' 버튼을 클릭합니다.
- 검증 포인트: 수정된 내용이 올바르게 반영되었는지 확인합니다.

4단계: 콘텐츠 삭제
- 삭제할 콘텐츠를 선택하고 '삭제' 버튼을 클릭합니다.
- 삭제 확인 메시지가 나타나면 '확인'을 클릭합니다.
- 검증 포인트: 삭제된 콘텐츠가 목록에서 사라졌는지 확인합니다.

4) 운영 체크리스트
- 일일 체크리스트:
  - 새로운 콘텐츠 업로드가 필요한지 확인.
  - 기존 콘텐츠의 최신성 검토.
- 주간 체크리스트:
  - 콘텐츠 사용 통계 검토.
  - 사용자 피드백 수집 및 검토.

5) 문제 해결 FAQ
- Q1: 로그인할 수 없습니다. 어떻게 해야 하나요?
  A1: 계정 정보가 올바른지 확인하고, 비밀번호 재설정을 시도하세요.
  
- Q2: 콘텐츠 업로드 중 오류가 발생했습니다. 
  A2: 파일 형식과 크기를 확인하고, 다시 시도해 보세요.
  
- Q3: 수정한 내용이 반영되지 않았습니다.
  A3: 저장 버튼을 클릭했는지 확인하고, 페이지를 새로 고침하세요.
  
- Q4: 삭제한 콘텐츠를 복구할 수 있나요?
  A4: 삭제된 콘텐츠는 복구할 수 없으므로 주의해야 합니다.
  
- Q5: 권한 요청이 승인되지 않았습니다. 
  A5: 인사팀에 문의하여 요청 사유를 확인하세요.

6) 보안/컴플라이언스
- 모든 직원은 개인 계정 정보를 보호해야 하며, 타인과 공유해서는 안 됩니다.
- 콘텐츠 업로드 시 저작권 및 개인정보 보호를 준수해야 합니다.
