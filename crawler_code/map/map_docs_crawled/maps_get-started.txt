Google Maps Platform 시작하기
이 페이지에서는 Google Maps Platform의 API 및 SDK [https://developers.google.com/maps/apis-by-platform?hl=ko]를 사용하기 위한 기본 필수 단계를 설명합니다.
빠른 시작
Google Cloud 콘솔을 사용하여 결제 계정 또는 프로젝트를 만든 적이 없는 경우 신규 사용자를 위한 Cloud 콘솔의 대화형 설정 환경에 연결되는 시작하기 버튼을 클릭하세요.
시작하기 [https://console.cloud.google.com/google/maps-apis/start?utm_source=Docs_GS_Button&ref=https%3A%2F%2Fdevelopers.google.com%2Fmaps%2F&utm_content=Docs_Central&hl=ko]
또는 다음 탭을 클릭하여 필요한 설정 단계를 완료할 수도 있습니다.



결제 계정 크레딧
결제 계정과 프로젝트를 만든 후 Google Cloud Platform 300달러 무료 체험판과 매월 제공되는 Google Maps Platform의 200달러 월간 크레딧을 받을 수 있습니다. 자세한 내용은 결제 계정 크레딧 [https://developers.google.com/maps/billing-credits?hl=ko]을 참고하세요.
다음 단계
설정이 완료된 후 Google Maps Platform API와 SDK를 사용할 수 있습니다. 자세한 내용은 관심 있는 API 또는 SDK의 개요, 개발자 가이드, 시작 가이드를 참고하세요. API 및 SDK는 Google Maps Platform 문서(제품별) [https://developers.google.com/maps/documentation?hl=ko] 및 Google Maps Platform API(플랫폼별) [https://developers.google.com/maps/apis-by-platform?hl=ko]에 나와 있습니다.
