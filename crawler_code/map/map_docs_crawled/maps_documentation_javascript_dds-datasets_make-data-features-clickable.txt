마우스 이벤트 처리
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/dds-datasets/make-data-features-clickable?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/dds-datasets/make-data-features-tappable?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/dds-datasets/make-data-features-clickable?hl=ko]
데이터 기능이 mousemove 및 click 이벤트에 응답하도록 하고 이를 사용하여 사용자 상호작용에 따라 기능의 모양을 변경합니다.
데이터 세트 레이어 이벤트 사용 설정
다음 단계에 따라 데이터 세트 레이어에서 이벤트를 사용 설정합니다.
등록할 각 이벤트의 데이터 세트 레이어에서 addListener() 함수를 호출하여 이벤트 알림에 사용할 데이터 세트 레이어를 등록합니다. 이 예시에서는 지도에도 리스너가 추가됩니다.
상호작용 유형에 따라 선택한 기능을 스타일 지정하는 이벤트 핸들러 코드를 추가합니다.
지형지물 스타일 함수를 사용하여 스타일을 적용합니다. 여기에 표시된 지형지물 스타일 함수는 상호작용 유형에 따라 조건부로 스타일을 적용합니다. 여기에는 세 가지 스타일이 정의되어 있습니다. 하나는 mousemove에서 테두리를 굵게 만드는 스타일이고, 하나는 click에서 배경을 변경하는 스타일이며, 하나는 기본 스타일입니다.
