프로젝트 역할 및 권한
지도 문서 [https://developers.google.com/maps?hl=ko]에 언급된 모든 Google Cloud 콘솔 역할과 해당 권한 및 기능을 나열한 표입니다.
셀이 공백인 경우 해당 열의 사용자 역할에 행의 권한이 부여되지 않았음을 나타냅니다. 사용자는 지정된 프로젝트에서 여러 Cloud 콘솔 역할을 가질 수 있습니다. 따라서 사용자가 특정 작업을 실행할 권한을 명시적으로 부여하는 역할을 가지고 있다면 사용자가 가진 다른 역할로 인해 특별히 해당 작업을 실행하지 못하는 경우는 없습니다.
Cloud 콘솔 역할에 대해 자세히 알아보기 [https://cloud.google.com/iam/docs/understanding-roles?hl=ko]

기술 지원 편집자 및 기술 지원 뷰어 역할은 현재 결제 케이스에 액세스할 수 있지만, 이는 향후 변경될 예정입니다. 적절한 결제 케이스 권한을 중단 없이 이용하려면 프로젝트 편집자 역할을 사용하세요.
지원 케이스를 만들어야 하는 결제 관리자는 이 해결 방법을 사용 [https://developers.google.com/maps/support?hl=ko#contact-billing-support-as-a-billing-admin]할 수 있습니다.
