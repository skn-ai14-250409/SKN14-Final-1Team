사용 가능한 경로 매트릭스 옵션
유럽 경제 지역 (EEA) 개발자
출발지와 도착지를 지정하는 것 외에도 경로에 사용할 수 있는 여러 옵션을 사용하여 경로 매트릭스를 맞춤설정할 수 있습니다. 이러한 옵션을 설정하는 방법에 관한 자세한 내용은 표에 링크된 주제를 참고하세요.
옵션 설명
여행 모드 [https://developers.google.com/maps/documentation/routes/reference/rest/v2/RouteTravelMode?hl=ko]
운전, 대중교통, 도보, 이륜차와 같은 이동 수단입니다.
참고: 도보, 자전거, 이륜차 경로는 베타 버전이며 명확한 인도, 보행자 전용 도로 또는 자전거 전용 도로가 누락되는 경우가 있습니다. 앱에 표시하는 모든 도보, 자전거, 이륜차 경로에 이 경고를 사용자에게 표시해야 합니다.
트래픽 데이터를 포함할지 여부 및 방법 지정 [https://developers.google.com/maps/documentation/routes/config_trade_offs-rm?hl=ko] 더 정확한 결과를 얻으려면 트래픽 데이터를 포함하고, 더 빠른 응답을 얻으려면 트래픽 데이터를 제외하세요.
경로 매트릭스의 통행료 계산 [https://developers.google.com/maps/documentation/routes/calculate_toll-rm?hl=ko] 경로 매트릭스에 경로의 예상 통행료 요금을 포함합니다.
현지화된 값 요청 [https://developers.google.com/maps/documentation/routes/localized-values-rm?hl=ko] 필드 마스크를 사용하여 경로와 동일한 방식으로 경로 매트릭스 응답에 대해 현지화된 텍스트를 요청합니다.
사용 가능한 경로 매트릭스 차량 유형 [https://developers.google.com/maps/documentation/routes/vehicles-rm?hl=ko] 경로 매트릭스의 차량 유형을 선택하는 추가 옵션입니다.
추가 옵션 출발 시간과 같은 추가 옵션을 지정하려면 요청 본문 [https://developers.google.com/maps/documentation/routes/reference/rest/v2/TopLevel/computeRouteMatrix?hl=ko#request-body]을 참고하세요.
