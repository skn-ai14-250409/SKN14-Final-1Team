Android용 Maps SDK 개요
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/overview?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/overview?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/overview?hl=ko]
Android용 Maps SDK를 사용해 Android 앱 [https://developer.android.com/?hl=ko](Google 지도 데이터, 지도 표시, 지도 동작 응답을 사용하는 Wear OS [https://developer.android.com/wear?hl=ko] 앱 포함)에 지도를 추가해 보세요. 또한 마커, 다각형, 오버레이를 지도에 추가하여 지도 위치에 대한 정보를 추가로 제공하고 사용자 상호작용을 지원할 수도 있습니다.
SDK는 Kotlin [https://developer.android.com/kotlin?hl=ko]과 Java 프로그래밍 언어를 모두 지원할 뿐만 아니라 고급 [https://developers.google.com/maps/documentation/android-sdk/overview?section=tools&hl=ko] 기능과 프로그래밍 기법을 사용할 수 있도록 라이브러리와 확장 프로그램을 추가로 제공합니다.

결제를 사용 설정하고 API 키를 가져오는 방법에 대한 세부정보입니다. 이 두 가지는 지도를 앱에 추가하기 위해 필요합니다.

Android용 Maps SDK를 사용할 수 있도록 기존 앱을 구성합니다.
