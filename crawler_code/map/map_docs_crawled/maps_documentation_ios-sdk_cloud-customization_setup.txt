시작하기 및 설정
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/setup?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/setup?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/setup?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/setup?hl=ko]
결제
클라우드 기반 지도 스타일 지정을 사용하려면 지도 ID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]가 필요합니다. Android용 Maps SDK, iOS용 Maps SDK, JavaScript에서 지도 ID를 사용하면 Dynamic Maps SKU [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#dynamic-maps-ess-sku]에 대한 요금이 부과됩니다. Maps Static API에서 지도 ID를 사용하면 Static Maps [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#static-maps-ess-sku] SKU에 대한 요금이 부과됩니다.
애플리케이션 및 웹사이트 요구사항
지도 스타일을 표시하려면 웹사이트 및 애플리케이션에 다음 플랫폼 버전이 필요합니다.
JavaScript 벡터 기반 지도: JavaScript Maps API 3.47 이상. 최신 버전을 설정하려면 Maps JavaScript API 설정 [https://developers.google.com/maps/documentation/javascript/load-maps-js-api?hl=ko]을 참고하세요.
JavaScript 래스터 기반 지도: JavaScript Maps API 3.49 이상. 최신 버전을 설정하려면 Maps JavaScript API 설정 [https://developers.google.com/maps/documentation/javascript/load-maps-js-api?hl=ko]을 참고하세요.
iOS: iOS Maps SDK 8.3.1 이상. 최신 버전을 설정하려면 Xcode 프로젝트 설정 [https://developers.google.com/maps/documentation/ios-sdk/config?hl=ko]을 참고하세요.
Android: Android SDK 버전 18.0.0 이상, 새로운 Android 지도 렌더기 [https://developers.google.com/maps/documentation/android-sdk/renderer?hl=ko] 및 Google Play 서비스 버전 23.33.16 이상 최신 버전을 설정하려면 Android 스튜디오 프로젝트 설정 [https://developers.google.com/maps/documentation/android-sdk/config?hl=ko]을 참고하세요.
참고: JavaScript 지도에서 대부분의 클라우드 스타일 지정 기능을 사용하려면 Maps JavaScript API 베타 채널 [https://developers.google.com/maps/documentation/javascript/versions?hl=ko#choosing-the-beta-channel]을 사용해 보세요.
시작할 준비가 되셨나요? 튜토리얼 [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/tut?hl=ko]을 참고하세요.
