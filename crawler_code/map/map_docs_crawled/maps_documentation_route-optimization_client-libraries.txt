Route Optimization API 클라이언트 라이브러리
이 페이지에서는 Route Optimization API용 클라이언트 라이브러리를 시작하는 방법을 설명합니다.
클라이언트 라이브러리에 대한 자세한 내용은 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.
클라이언트 라이브러리 설치
인증 설정
클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보 (ADC) [https://cloud.google.com/docs/authentication/application-default-credentials?hl=ko]를 사용하여 인증합니다. ADC 설정에 대한 자세한 내용은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공 [https://cloud.google.com/docs/authentication/provide-credentials-adc?hl=ko]을 참고하세요. 클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증 [https://cloud.google.com/docs/authentication/client-libraries?hl=ko]을 참조하세요.
